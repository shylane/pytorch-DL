# pytorch-DL
 Exploring deep learning models using pytorch

## Prerequisites
- Python 3.10+
- Pytorch 2.2.1+
- pytorch-grad-cam 1.5.0

## Training
```
# Ensure installing grad-cam
%pip install grad-cam

# Start training with: 
python main.py

# You can manually resume the training with: 
python main.py --resume --lr=0.01
```
